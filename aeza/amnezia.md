# О чём эта статья?

Эта статья о том как восстановить работработоспособность приложения Amnezia на серверах Aeza. Она будет предствалена в виде скриншотов с некоторыми пояснениями.
Эта инструкция подойдёт для всех платформ где есть Amnezia.

После этого вы можете пробовать подключиться к VPN.

Если вам AmneziaWG нужен из-за роутера то для keenectic Vless вы можете настроить следуя этой инструкции: https://github.com/Corvus-Malus/XKeen
Для Mikrotik Vless вы можете настроить следуя этой инструкции: https://ruii.ru/xray-vless-reality-na-mikrotik
Остальные роутеры можно перепрошить на OpenWRT и настроить Vless следуя этой инструкции: https://podkop.net/docs/install/

Действия по перепрошивке и настройки роутера выполняются на ваш страх и риск!
